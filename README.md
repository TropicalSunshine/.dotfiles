Hello welcome to my dot files managed by home-manager, a home folder project built on the nix package manager ! 

to begin install nix here - https://nixos.org/download/

then install and setup home-manager - https://nix-community.github.io/home-manager/index.xhtml#sec-install-standalone

```
home-manager switch --flake .#aarch64-darwin
home-manager switch --flake .#x86_64-linux
```
